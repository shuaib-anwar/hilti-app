# Hilti-App
Manage Profile Functionalty for Hilti


## Commands to run

### Add ionic in environment variable
```
npm install -g cordova ionic
```
### Add cordova in environment variable
```
npm install -g cordova
```

### Install Node Packages
```
npm install
```

### Run on Browser
```
ionic serve
```

### Add iOS Platform
```
ionic platform add ios
```

### Build For iOS
```
ionic build ios
```
